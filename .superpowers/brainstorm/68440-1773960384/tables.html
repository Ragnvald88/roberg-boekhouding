Better Tables — What matters most?

Pick which table improvements would make the biggest difference to you

1

Inline actions — no hidden menus

Status changes (betaald/onbetaald) as toggle buttons directly in the row, not buried in a three-dot menu. Delete stays in menu (destructive = hidden). Edit on row click or double-click.

2

Smart column formatting

All money columns right-aligned with tabular numbers and consistent € formatting. Dates always Dutch format. Status as colored chips. Negative amounts in red. Totals row at bottom with bold styling.

3

Expandable row details

Click a factuur row to expand and see its werkdagen inline. Click a werkdag to see the linked factuur. No need to navigate to another page — the relationship is visible in-place.

4

Smart filter presets

Instead of separate dropdowns for year/month/klant/status, add quick-filter chips: "Ongefactureerd", "Openstaand", "Deze maand", "Vorige maand". One click to filter, one click to clear.

5

Better bulk actions

Select multiple werkdagen → "Factureer" button appears with a preview of the invoice right there. No page navigation. Select multiple facturen → "Markeer betaald" in one click.

6

Visual status indicators

Row background colors indicating status (like the bank page already does). Overdue invoices highlighted in warm colors. Subtle left-border color per status. Makes scanning the table instant.
